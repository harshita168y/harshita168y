## Featured projects

### [Chatzzz — AI-moderated real-time chat platform](https://github.com/harshita168y/webchat)
A group chat app built for anime communities with context-aware AI moderation. Uses **SignalR** for real-time messaging, **OpenAI** to evaluate messages with conversation history, and a violation tracking system that auto-bans repeat offenders. Moderation decisions are logged with score, category, and reasoning for full auditability.

`ASP.NET Core 8` `SignalR` `PostgreSQL` `OpenAI` `React Native` `TypeScript` `Entity Framework`

---

### [Video RAG Chatbot — face recognition + memory](https://github.com/harshita168y/Video-chatbot)
A multimodal AI assistant that recognises faces via webcam, remembers past conversations using **Pinecone vector storage**, and responds naturally through Ollama or OpenAI. Built on FastAPI and WebSockets with optional speech-to-text and text-to-speech support.

`FastAPI` `Python` `InsightFace` `Pinecone` `RAG` `Ollama` `OpenAI` `WebSockets`

---

### [AI Email Automation — human-in-the-loop workflow](https://github.com/harshita168y/AI-Powered-Email-Automation-Workflow-)
An intelligent Gmail automation system that drafts AI replies to incoming emails and routes them through a human review step — approve, edit, or reject — before sending. Built with n8n, Gmail API OAuth2, and Gemini.

`n8n` `Gmail API` `Gemini API` `Webhook automation`

---

### [Exercise Monitoring — real-time pose estimation on Android](https://github.com/harshita168y/Exercise-Monitoring-and-Pose-Estimation)
An Android app using **MediaPipe BlazePose** for real-time exercise tracking. Counts reps, detects movement stages, and gives instant audio feedback for Hammer Curl, Wall Sit, and Front Arm Raise — all running on-device with no noticeable latency.

`Kotlin` `Android` `MediaPipe BlazePose` `Computer Vision`
